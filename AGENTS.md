# AGENTS.md

**ai-tracker** — a zero-dependency Python package (`aitracker/`, stdlib only) that serves a local dashboard for AI coding sessions. The browser UI lives as real files in `aitracker/web/` and is inlined at serve time; each AI tool plugs in as a `Provider` under `aitracker/providers/`.

- **Run / test:** `make serve` (start on :8787), `make stop`, `make check` (= `python -m unittest discover -s tests`, must print `selfcheck ok`). Or `python -m aitracker`, or install with `uv` → `ai-tracker`.
- **Restart to see changes** — the page is assembled from `web/*` at startup; only `flags.json`/`titles.json` are read live.
- **Edit `aitracker/` + `web/`, never `dist/tracker.py`** (generated by `make bundle`).
